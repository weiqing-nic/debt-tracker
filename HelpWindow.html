1. Introduction

Debt Tracker (DT) is for those who prefer to use a desktop app for tracking the user’s loans. DT has a mixed usage of Command Line Interface (CLI) and a Graphical User Interface (GUI). For more basic commands, one may use CLI, but DT has also simplified potentially longer commands through the use of GUI. DT can keep a record of your loans and debts as fast as other apps, if not faster!

2. Features

2.1. Viewing help : help

Invokes the user guide for handy reference when typing commands.

2.2. Adding a person: add

Adds a person to the Debt Tracker

2.3. Adding interest rate to a debt: interest

Adds an interest rate to a specific recorded debt for app to track.

2.4. Hiding a private contact : hide

Makes contacts tagged as private invisible. Can be toggled.

2.5. Editing a person : edit

Edits an existing person in the address book.

2.6. Locating persons by name: find

Finds persons whose names contain any of the given keywords.

2.7. Deleting a person : delete

Deletes the specified person from the address book.

2.8. Removing a debt : remove

Remove away a number of debts from one person recorded in the DT using click-and-drag. The number of debts removed is up to the user’s selection.

2.9. Selecting a person : select

Selects the person identified by the index number used in the displayed person list.

2.10. Listing entered commands : history

Lists all the commands that you have entered in reverse chronological order.

2.11. Listing all recorded expenses: log

Creates a ledger of all the transactions that have been recorded.

2.12. Undoing previous command : undo

Restores the address book to the state before the previous undoable command was executed.

2.13. Redoing the previously undone command : redo

Reverses the most recent undo command.

2.14. Clearing all entries : clear

Clears all entries from the Debt Tracker.

2.15. Exiting the program : exit

Exits the program.

2.16. Saving the data

Debt Tracker data is saved in the hard disk automatically after any command that changes the data.
